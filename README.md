# TSMFM Dashboard — بوصلة الدراسة

## كيفاش تشغل المشروع محلياً

1. تأكد بلي عندك Node.js مثبت (نزله من https://nodejs.org إلى ماكانش).
2. افتح Terminal / CMD فالفولدر ديال المشروع.
3. دير:
   ```
   npm install
   npm run dev
   ```
4. حل الرابط لي غادي يبان (مثلا http://localhost:5173).

## نشر التطبيق أونلاين (Vercel)

1. دير حساب فـ https://vercel.com (تقدر تدخل بـ GitHub مباشرة).
2. زيد المشروع (بصيغة drag & drop ولا عبر GitHub).
3. Vercel غادي يكتشف بلي هو مشروع Vite ويبنيه وحدو.
4. تحصل على رابط دائم بحال: `tsmfm-dashboard.vercel.app`

## ملاحظة مهمة بخصوص تبويب "المساعد الذكي" (AI Tab)

الكود الأصلي كان معمول باش يخدم جوا Claude.ai فين الـ API key متدارة تلقائياً.
بره Claude.ai، الاتصال المباشر بـ `api.anthropic.com` من المتصفح **ما غاديش يخدم**
(محجوب بسبب CORS + خاصو API key). التطبيق ما غاديش يهبط، غير هاد التبويب بالضبط
غادي يعطي رسالة خطأ "وقع خطأ فالاتصال".

باش تخدم هاد الميزة بره Claude، خاصك تدير:
- Backend بسيط (مثلا Vercel Serverless Function) يحتفظ بـ API key ديالك بأمان
  ويدير الطلب لـ Anthropic API نيابة عن التطبيق.

نقدر نبنيوها ليك من بعد إلى بغيتي.
